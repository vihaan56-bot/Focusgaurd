Installation Instructions

  1. Download the APK using the button above.
  2. Open the downloaded FocusGuard-release.apk file on your Android phone.
  3. If prompted by Android, allow installation from this source.
  4. Install FocusGuard on your device.
  5. Open FocusGuard.
  6. Grant Usage Access permission when requested.
  7. Test the Phase 1 dashboard, app limits, and focus mode functionality.
🔒 Privacy Guarantee

FocusGuard Phase 1 processes usage information locally on your Android device. No analytics, tracking, or advertising SDKs are included.
